# SD01 Friendships Begin Collector

A fan-made collector tool for the **Kayou My Little Pony Card Game SD01 Friendships Begin** set.

This project allows collectors to track their collection, view card images, manage duplicates, track starter deck cards, and export collection data.

## Features

### Collection Tracking

- Mark cards as owned
- Track card quantities
- Track duplicate cards
- Wishlist support
- Collection completion statistics

### Card Information

- Card images
- Card names
- Card numbers
- Card rarities
- Pull odds reference

### SD01 Deck Support

Includes support for cards from:

- Twilight Sparkle Starter Deck
- Fluttershy Starter Deck
- Pinkie Pie Starter Deck
- Applejack Starter Deck
- Rainbow Dash Starter Deck
- Rarity Starter Deck

### Collector Tools

- CSV checklist export
- JSON backup system
- JSON restore system
- Print-friendly binder view
- Search system
- Rarity filtering
- Deck filtering

### Analytics

Google Analytics integration has been added to help improve future versions.

Analytics Page Name:
